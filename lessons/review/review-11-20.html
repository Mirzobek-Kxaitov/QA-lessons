Confirmation va Maintenance Testing

Test Texnikalar

BVA (Boundary Value Analysis) — chegaradagi qiymatlarni test qilish (17, 18, 65, 66). EP (Equivalence Partitioning) — bir xil natija beradigan qiymatlar guruhi. Decision Table — shartlar kombinatsiyasi. State Transition — holat o'tishi diagrammasi.

Test Plan

Test Plan — test jarayonini rejalashtirish hujjati. Asosiy bo'limlar: Scope (nima test qilinadi), Out of Scope (nima qilinmaydi), Rislar, Jadval, Resurslar, Entry/Exit Criteria. Entry Criteria — test boshlanish sharti. Exit Criteria — test tugash sharti.

Jira — Bug Tracking

Jira — loyiha va bug boshqaruv tizimi. Issue Types: Bug, Task, Story, Epic. Bug yozish: Title, Steps, Expected, Actual, Severity, Priority, Screenshot. JQL (Jira Query Language) — filtrlash: assignee = currentUser(). Kanban taxtasi — vazifalar ko'rinishi.

16

TestRail — Test Management

TestRail — test caselarni boshqarish tizimi. Ierarxiya: Project → Suite → Section → Test Case. Test Run — test caselarni bajarish sessiyasi. Natijalar: Pass ✅, Fail ❌, Blocked 🚫, Untested ⏸️, Retest 🔁. Jira bilan integratsiya — bug va test case bog'lash.

17

Postman va API Testing

API — dasturlar orasidagi ko'prik. REST API: GET (o'qish), POST (yaratish), PUT (yangilash), DELETE (o'chirish). Status Codes: 200 OK, 201 Created, 400 Bad Request, 401 Unauthorized, 404 Not Found, 500 Server Error. JSON — eng mashhur ma'lumot formati.

APIRESTPostmanStatus Codes

Automation Asoslari

Automation — testlarni kompyuter bajaradi. Test Pyramid: Unit 70%, API 20%, UI/E2E 10%. Qachon automation: takrorlanadigan, regression, performance testlar. Qachon manual: yangi feature, UX, bir martalik. ROI — 5+ marta takrorlanadigan testlar uchun foydali.

Selenium WebDriver (Python)

Selenium WebDriver — kod bilan brauzer boshqarish. Asosiy buyruqlar: driver.get(url), find_element(By.ID, "id"), click(), send_keys(). Locator tartibi: By.ID → By.NAME → By.CSS → By.XPATH. Explicit Wait — WebDriverWait bilan eng ishonchli kutish.

Playwright — Zamonaviy Automation

Playwright — Microsoft, 2020. Asosiy farqi: Auto-wait — element tayyor bo'lgunicha o'zi kutadi (Selenium'da Wait yozish kerak edi). Asosiy buyruqlar: page.goto(url), page.fill(sel, txt), page.click(sel). Codegen — harakatlarni yozib kod chiqaradi. Yangi loyiha uchun tavsiya etiladi.
